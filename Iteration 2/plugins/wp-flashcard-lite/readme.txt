== Description ==

Easily add great looking and responsive flashcards to your WordPress site. Simple and fun to use, WP Flashcard LITE allows you to insert flashcards via a shortcode anywhere in a post or page. Add a flashcard set to your website with up to 10 terms and definitions and place it anywhere you like.

== Installation ==
Starting with WP Flashcard LITE consists of just three easy steps: installing, activating, and adding a flashcard set to your site. 

### INSTALL WP FLASHCARD LITE FROM WITHIN WORDPRESS

1. Visit the plugins page within your dashboard and select ‘Add New’
2. Search for ‘WP Flashcard LITE’
3. Activate WP Flashcard LITE from your Plugins page
4. Go to ‘after activation’ below.

### INSTALL WP FLASHCARD LITE MANUALLY

1. Upload the ‘wp-flashcard-lite’ folder to the /wp-content/plugins/ directory
2. Activate the WP Flashcard LITE plugin through the ‘Plugins’ menu in WordPress
3. Go to ‘after activation’ below.

### AFTER ACTIVATION

1. You should see a new menu item called ‘Flashcards’
2. Click to ‘Add New’
3. Enter your card front and card back until you have your set completed, up to 10 cards
4. Click to publish
5. Copy and paste the provided shortcode to insert the set anywhere in a post or page on your site
